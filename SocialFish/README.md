### CLONE
```
git clone https://github.com/UndeadSec/SocialFish.git
```

### RUNNING

```
cd SocialFish
```

```
sudo apt-get install python3-pip -y
```

```
sudo pip3 install -r requirements.txt
```

```
python3 SocialFish.py
```
